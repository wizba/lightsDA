## Description

LightNode Rest-API build with NEST Framework


## Arduino section
Make sure the Arduino device is plug and running


## Install Nest globally

```bash
sudo npm i -g @nestjs/cli
```


## Run LightNode

```bash
# install modules
$ npm install

# development watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
