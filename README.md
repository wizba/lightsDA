## Description

node Rest-API build with NEST Framework


## Arduino section
Make sure the Arduino device is plug and running


## Install Nest globally

```bash
sudo npm i -g @nestjs/cli
```


## Run LightNode

```bash
# switch to node directory
$ cd node

# install node modules
$ npm install

# run the project
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Description

light mobile app build with IONIC 4 Framework


## Install Ionic globally

```bash
sudo npm install -g ionic
```


## Run light

```bash
# switch to light directory
$ cd light

# install modules
$ npm install

# run ionic project
$ ionic serve

# run ionic, test devices on the browser 
$ ionic serve -l

# run ionic on dev-app (phone)
$ ionic serve -devapp

# run ionic on an android device (check documentation for setting up)
$ ionic cordova run android
```
